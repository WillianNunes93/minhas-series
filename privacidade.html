Política de Privacidade

Última atualização: 15 de julho de 2026

O que é o Minhas Séries

Minhas Séries é um projeto pessoal para catalogar e acompanhar séries de TV, criado para uso próprio e compartilhado com familiares e amigos. Não é um produto comercial, não exibe anúncios e não vende dados de nenhum tipo.

Quais dados coletamos

Para criar sua conta e usar o app, guardamos:

  • E-mail e senha, usados para login (gerenciados pelo Firebase Authentication, do Google).
  • Nome de exibição e avatar (um emoji), que você escolhe no seu perfil.
  • As séries que você adiciona: nome, status, nota, episódios assistidos e distribuidoras — para montar sua lista pessoal.
  • Conexões de amizade dentro do app, se você adicionar outros usuários como amigos.

Não coletamos localização, contatos do celular, câmera, nem nenhum dado do seu dispositivo além do necessário para o app funcionar.

Como os dados são usados

Servem só para o funcionamento do app: mostrar sua lista de séries, calcular estatísticas, permitir que amigos vejam as séries que você marcou como favoritas ou em andamento (só se vocês forem amigos aceitos um do outro), e enviar notificações sobre lançamentos.

Com quem compartilhamos dados

Os dados ficam armazenados no Firebase (Google Cloud). Buscamos informações públicas de séries (nome, sinopse, pôster, elenco, distribuidoras) na API do TMDB — essas buscas não enviam seus dados pessoais para o TMDB, só o termo pesquisado. Não vendemos, alugamos ou compartilhamos seus dados com terceiros para fins de publicidade.
